# powerbi-api-paises
Dashboard em Power BI usando dados da API RestCountries e carregados via Python + PostgreSQL.

Projeto simples mostrando ingestão de dados da API RestCountries com Python,
armazenamento em PostgreSQL e visualização final em Power BI.

### tecnologias:

- Python (requests, psycopg2)
- PostgreSQL
- Power BI
